POS INTEGRATIONS SIMULATOR

A dashboard to simulate the ordering APIs of UrbanPiper.

Below are the features you can simulate using this dashboard,

  • Order Relay Webhook — Order Relay simulation of aggregators with various scenarios such as, aggregator & merchant discounts, item and order level taxes, item and order level charges, taxes on charges, etc
  • Rider Status Update Webhook — Simulation of rider status such as assigned, at-store, delivered, etc
  • Swiggy Cancellation Status Webhook — A webhook simulation for Swiggy order cancellation request.
  • Store Actions Webhook — A store actions API webhook simulation for different aggregators
  • Item Actions Webhook — A item actions API webhook simulation for different aggregators
